Mysql-docs | A simple tool to generate docs for mysql
######################################################
.. image:: https://github.com/mggger/mysql-docs/blob/master/images/shot.png


1. *安装依赖的第三方库*  **pip3 install -r requirements.txt**
2. *在config.yml配置mysql相关, 指定生成.rst文件夹*
3. *python3 main.py*
